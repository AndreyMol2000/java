# Мультимедиа в HTML: видео, аудио, iframe

## Краткая информация
HTML позволяет добавлять на страницу мультимедиа-контент: видео, аудио и встроенные страницы через `<iframe>`. Эти элементы делают сайт более интерактивным и визуально насыщенным.

---

## Пример

```html
<!-- Видео -->
<video width="320" height="240" controls>
  <source src="video.mp4" type="video/mp4">
  Ваш браузер не поддерживает видео.
</video>

<!-- Аудио -->
<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
  Ваш браузер не поддерживает аудио.
</audio>

<!-- Встроенная страница -->
<iframe src="https://example.com" width="600" height="400"></iframe>
Подробная информация
Видео <video>
Атрибуты:

controls — отображает элементы управления (play, pause, громкость).

autoplay — воспроизведение сразу после загрузки страницы.

loop — повторное воспроизведение.

muted — звук выключен по умолчанию.

Тег <source> позволяет указать несколько форматов видео для совместимости с разными браузерами.

Аудио <audio>
Атрибуты аналогичны <video> (controls, autoplay, loop, muted).

Тег <source> поддерживает различные форматы аудио (mp3, ogg, wav).

Встроенные страницы <iframe>
Позволяет встроить на страницу внешнюю HTML-страницу.

Атрибуты:

src — ссылка на внешний ресурс.

width и height — размеры окна.

allowfullscreen — разрешение на полноэкранный режим.

Важно: некоторые сайты могут блокировать показ через <iframe> по настройкам безопасности.

Пример с расширением
html
Копировать код
<h2>Мультимедиа на странице</h2>

<section>
  <h3>Видеоурок</h3>
  <video width="480" height="270" controls autoplay muted loop>
    <source src="lesson.mp4" type="video/mp4">
    Ваш браузер не поддерживает видео.
  </video>
</section>

<section>
  <h3>Подкаст</h3>
  <audio controls>
    <source src="podcast.mp3" type="audio/mpeg">
    Ваш браузер не поддерживает аудио.
  </audio>
</section>

<section>
  <h3>Встраивание внешнего сайта</h3>
  <iframe src="https://example.com" width="800" height="600" allowfullscreen></iframe>
</section>
Объяснение
<video> и <audio> позволяют воспроизводить мультимедиа без установки плагинов.

Использование нескольких <source> обеспечивает поддержку разных форматов для всех браузеров.

<iframe> полезен для:

Встраивания карт (Google Maps)

Видео с YouTube

Внешних виджетов или приложений

Атрибуты autoplay, loop, muted делают контент более динамичным и интерактивным.

Мультимедиа повышает вовлечённость пользователей и делает сайт современным и привлекательным.
