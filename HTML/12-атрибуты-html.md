# Атрибуты и глобальные атрибуты в HTML

## Краткая информация
Атрибуты позволяют задавать **характеристики и свойства элементов** в HTML.  
Существуют **специфические атрибуты** для каждого тега и **глобальные атрибуты**, которые работают почти со всеми элементами.

---

## Пример использования атрибутов

```html
<p id="paragraph1" class="text-muted" title="Подсказка при наведении">Пример текста</p>
id — уникальный идентификатор элемента.

class — класс для стилизации и группировки элементов.

title — текст подсказки при наведении курсора.

Подробная информация
Глобальные атрибуты
Могут использоваться практически со всеми тегами:

id — уникальный идентификатор элемента.

class — один или несколько классов для стилизации и JS.

title — всплывающая подсказка.

style — встроенные CSS-стили.

hidden — скрывает элемент на странице.

data-* — пользовательские атрибуты для хранения данных (data-user="123").

tabindex — порядок перехода при навигации через клавиатуру.

contenteditable — делает элемент редактируемым пользователем.

Специфические атрибуты
Каждый тег может иметь свои атрибуты. Например:

<a href="url" target="_blank" rel="noopener">Ссылка</a>

href, target, rel

<img src="img.jpg" alt="Описание" width="200" height="150">

src, alt, width, height

<input type="text" name="username" placeholder="Имя">

type, name, placeholder, value, required, maxlength

Пример комбинированного использования
html
Копировать код

Объяснение
Атрибуты определяют поведение и свойства элементов.

Глобальные атрибуты повышают универсальность и контроль над элементами.

Использование id и class важно для CSS и JavaScript, так как позволяет находить элементы.

Пользовательские атрибуты data-* дают возможность хранить дополнительные данные без изменения структуры HTML.

Атрибуты делают HTML более семантичным, управляемым и интерактивным.
