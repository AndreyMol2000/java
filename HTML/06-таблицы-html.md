# Таблицы в HTML

## Краткая информация
Таблицы позволяют структурировать данные в виде строк и колонок. Основные теги: `<table>`, `<tr>`, `<th>`, `<td>`.  

---

## Пример

```html
<table border="1">
  <tr>
    <th>Имя</th>
    <th>Возраст</th>
    <th>Город</th>
  </tr>
  <tr>
    <td>Андрей</td>
    <td>25</td>
    <td>Москва</td>
  </tr>
  <tr>
    <td>Елена</td>
    <td>30</td>
    <td>Санкт-Петербург</td>
  </tr>
</table>
Подробная информация
<table> — создаёт таблицу.

<tr> — создаёт строку таблицы.

<th> — создаёт заголовочную ячейку (по умолчанию текст жирный и по центру).

<td> — создаёт обычную ячейку таблицы.

Атрибуты

border — рамка таблицы.

colspan — объединение нескольких колонок.

rowspan — объединение нескольких строк.

align, valign — выравнивание текста в ячейках.

Пример с объединением ячеек
html
Копировать код
<table border="1">
  <tr>
    <th>Имя</th>
    <th colspan="2">Информация</th>
  </tr>
  <tr>
    <td>Андрей</td>
    <td>25 лет</td>
    <td>Москва</td>
  </tr>
  <tr>
    <td>Елена</td>
    <td rowspan="2">30 лет</td>
    <td>Санкт-Петербург</td>
  </tr>
  <tr>
    <td>Иван</td>
    <td>Новосибирск</td>
  </tr>
</table>
Объяснение
Таблицы полезны для структурированных данных, таких как расписания, списки, отчёты.

<th> делает текст заголовочным, <td> — обычным.

colspan и rowspan позволяют объединять ячейки для более сложной структуры таблицы.

Атрибут border можно заменить на CSS для более гибкой стилизации.

Использование таблиц для макета страницы устарело; сейчас их используют только для данных.

Таблицы делают информацию структурированной и читаемой для пользователя.
