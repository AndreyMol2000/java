# Семантические теги HTML

## Краткая информация
Семантические теги HTML используются для улучшения структуры страницы и повышения её доступности. Они помогают браузерам, поисковым системам и вспомогательным технологиям (например, скринридерам) лучше понимать содержимое страницы.

---

## Пример

```html
<header>
  <h1>Главная страница</h1>
</header>

<nav>
  <ul>
    <li><a href="#home">Главная</a></li>
    <li><a href="#about">О нас</a></li>
    <li><a href="#contact">Контакты</a></li>
  </ul>
</nav>

<main>
  <article>
    <h2>Новости</h2>
    <p>Последние обновления на сайте.</p>
  </article>
  <aside>
    <h3>Боковая панель</h3>
    <p>Ссылки и дополнительные материалы.</p>
  </aside>
</main>

<footer>
  <p>Все права защищены © 2025</p>
</footer>
Подробная информация
<header> — шапка сайта или секции, обычно содержит логотип, заголовок, навигацию.

<nav> — блок с навигационными ссылками.

<main> — основной контент страницы (должен быть только один на странице).

<article> — самостоятельный блок контента (новости, статьи, публикации).

<aside> — побочный контент, не связанный напрямую с основным содержимым (боковые панели, виджеты).

<footer> — нижний колонтитул сайта или секции, обычно содержит авторские права, ссылки, контакты.

Другие семантические теги

<section> — раздел страницы с тематическим контентом.

<figure> и <figcaption> — для изображений с подписями.

<mark> — выделенный текст.

<time> — дата или время.

<address> — контактная информация.

Объяснение
Семантические теги улучшают структуру и доступность страницы.

<article> используется для содержимого, которое можно отдельно публиковать.

<aside> — дополнительная информация, не обязательно связанная с основной статьей.

<figure> и <figcaption> помогают связывать изображения с описанием.

<time> с атрибутом datetime позволяет машинам (поисковикам, календарям) правильно распознавать дату.

Использование семантических тегов повышает SEO и делает сайт удобным для всех пользователей.
