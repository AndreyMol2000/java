# Ссылки и кнопки в HTML

## Краткая информация
Ссылки и кнопки — ключевые элементы навигации и взаимодействия на сайте.  
- **Ссылки** `<a>` — переход на другой URL или якорь на той же странице.  
- **Кнопки** `<button>` — выполняют действия: отправка формы, JavaScript-функции, интерактивные элементы.  

---

## Пример ссылки

```html
<a href="https://example.com" target="_blank" title="Перейти на сайт Example">Перейти на Example</a>
href — адрес ссылки.

target="_blank" — открытие в новой вкладке.

title — подсказка при наведении.

Пример кнопки
html
Копировать код
<button onclick="alert('Вы нажали кнопку!')">Нажми меня</button>
Атрибут onclick позволяет запускать JavaScript при клике.

Кнопки <button> можно стилизовать через CSS.

<input type="button"> — альтернатива <button>, но <button> гибче.

Подробная информация
Ссылки <a>
Могут вести на внешние страницы, внутренние страницы или якоря (#section1).

Атрибуты:

href — адрес.

target — _blank, _self, _parent, _top.

rel — nofollow, noopener, noreferrer для SEO и безопасности.

Внутри ссылки можно вставлять текст, изображения или другие элементы.

Кнопки <button> и <input type="button">
<button>:

Внутри можно вставлять HTML (текст, иконки, изображения).

Тип кнопки: type="button", type="submit", type="reset".

<input type="button">:

Только текст через атрибут value.

Не поддерживает вложенные HTML-элементы.

Для действий на странице используют JavaScript: onclick, onmouseover, onkeydown.

Пример расширенного использования
html
Копировать код
<h2>Навигация и кнопки</h2>

<nav>
  <a href="#home">Главная</a> |
  <a href="#about">О нас</a> |
  <a href="https://example.com" target="_blank" rel="noopener">Сайт Example</a>
</nav>

<section>
  <button onclick="alert('Вы нажали кнопку!')">Нажми меня</button>
  <button type="submit" form="form1">Отправить форму</button>
</section>

<form id="form1">
  <input type="text" name="name" placeholder="Введите имя">
</form>
Объяснение
Ссылки создают навигацию и связи между страницами и сайтами.

Атрибуты target и rel помогают контролировать поведение и безопасность.

Кнопки позволяют делать сайт интерактивным, выполнять действия через JavaScript или отправлять формы.

Использование <button> предпочтительнее <input type="button">, так как он гибче для стилизации и вставки HTML.

Ссылки и кнопки — основа пользовательского взаимодействия на сайте.
