# Базовые теги HTML

## Краткая информация
Базовые теги HTML используются для создания основной структуры страницы и отображения текста. К ним относятся заголовки, абзацы, списки, ссылки и изображения.

---

## Пример

```html
<h1>Заголовок уровня 1</h1>
<p>Это абзац текста.</p>
<a href="https://example.com">Ссылка на сайт</a>
<img src="image.jpg" alt="Описание изображения">
Подробная информация
Заголовки (<h1>–<h6>) — обозначают заголовки с разной степенью важности.

Параграфы (<p>) — основной блок текста.

Списки

Маркированные (<ul> + <li>)

Нумерованные (<ol> + <li>)

Ссылки (<a href="URL">) — переход на другие страницы или сайты.

Изображения (<img src="URL" alt="Описание">) — отображение картинок.

Комментарии (<!-- комментарий -->) — для заметок в коде, не отображаются на странице.

Пример (с расширением)
html
Копировать код

<p>Это мой первый абзац.</p>

<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>

<p>Посетите <a href="https://example.com" target="_blank">мой сайт</a> для больше информации.</p>

<img src="logo.png" alt="Логотип сайта">
Объяснение
<h1> используется для главного заголовка страницы, <h2>–<h6> — для подзаголовков.

<ul> создает маркированный список, <ol> — нумерованный.

<a> позволяет перейти по ссылке; атрибут target="_blank" открывает ссылку в новой вкладке.

<img> вставляет изображение, атрибут alt обеспечивает описание для пользователей и SEO.

Эти теги составляют основу структуры контента страницы, без них нельзя создавать информативные и понятные страницы.
