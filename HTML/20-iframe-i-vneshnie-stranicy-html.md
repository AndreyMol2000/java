# Фреймы и подключение внешних страниц в HTML

## Краткая информация
HTML позволяет **встраивать внешние страницы или ресурсы** в текущий документ с помощью тега `<iframe>`.  

- `<iframe>` — Inline Frame, вставка одного HTML-документа в другой.  
- Используется для: карт, видео, виджетов, сторонних сервисов.

---

## Пример базового iframe

```html
<iframe src="https://www.example.com" width="600" height="400" title="Пример сайта">
  Ваш браузер не поддерживает iframe.
</iframe>
src — URL внешней страницы или ресурса.

width и height — размеры фрейма.

title — описание для доступности.

Текст внутри <iframe> отображается, если браузер не поддерживает тег.

Пример YouTube видео через iframe
html
Копировать код
<iframe width="560" height="315" 
        src="https://www.youtube.com/embed/dQw4w9WgXcQ" 
        title="YouTube video player" 
        frameborder="0" 
        allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture" 
        allowfullscreen>
</iframe>
frameborder="0" — убирает рамку.

allowfullscreen — позволяет разворачивать видео на весь экран.

Атрибут allow управляет доступными действиями (автозапуск, гироскоп и т.д.).

Подключение внешних ресурсов (CSS, JS, шрифты)
html
Копировать код
<!-- CSS -->
<link rel="stylesheet" href="styles.css">

<!-- JS -->
<script src="script.js" defer></script>

<!-- Шрифты -->
<link href="https://fonts.googleapis.com/css2?family=Roboto&display=swap" rel="stylesheet">
<link> — подключение стилей или шрифтов.

<script> — подключение внешнего JS. Атрибут defer откладывает выполнение до полной загрузки HTML.

Подключение внешних ресурсов позволяет разделять код и использовать сторонние библиотеки.

Подробная информация
iframe
Встраивает независимый документ в страницу.

Можно применять стили CSS через width, height, border и позиционирование.

sandbox — ограничивает возможности iframe (например, отключает скрипты, формы, плагины).

html
Копировать код
<iframe src="example.html" sandbox></iframe>
Подключение внешних ресурсов
CSS через <link> улучшает поддержку стилей и позволяет подключать фреймворки (Bootstrap, Tailwind).

JS через <script> позволяет подключать библиотеки (jQuery, React) или собственные скрипты.

Шрифты через <link> подключаются с Google Fonts или других сервисов.

Объяснение
<iframe> делает сайт более модульным и позволяет использовать сторонние ресурсы без копирования контента.

Подключение внешних CSS/JS/шрифтов делает проект масштабируемым и удобным в поддержке.

Атрибуты безопасности (sandbox, allow) предотвращают нежелательные действия со стороны стороннего контента.

В современных сайтах <iframe> часто используют для виджетов, видео, карт и интеграции с внешними сервисами.
