# Формы и пользовательский ввод в HTML

## Краткая информация
Формы позволяют пользователю вводить данные на веб-странице и отправлять их на сервер. Основные элементы форм: текстовые поля, кнопки, чекбоксы, радиокнопки, выпадающие списки и текстовые области.

---

## Пример

```html
<form action="/submit" method="POST">
  <label for="username">Имя:</label>
  <input type="text" id="username" name="username" placeholder="Введите ваше имя" required>
  
  <label for="age">Возраст:</label>
  <input type="number" id="age" name="age" min="0" max="120">
  
  <button type="submit">Отправить</button>
</form>
Подробная информация
Тег <form>

Атрибут action — URL, куда отправляются данные.

Атрибут method — метод отправки (GET или POST).

Поля ввода <input>

type="text" — обычный текст.

type="number" — числовой ввод.

type="password" — скрытый текст.

type="email" — проверка ввода email.

type="checkbox" — чекбокс.

type="radio" — радиокнопка.

type="submit" — кнопка отправки формы.

Текстовая область <textarea> — для длинного текста.

Выпадающий список <select>

html

<select name="color">
  <option value="red">Красный</option>
  <option value="green">Зелёный</option>
  <option value="blue">Синий</option>
</select>
Метки <label> — связывают текст с полем ввода через атрибут for.

Обязательные поля — атрибут required заставляет пользователя заполнить поле.

<form action="/register" method="POST">
  <fieldset>
    <legend>Регистрация пользователя</legend>
    
    <label for="username">Имя:</label>
    <input type="text" id="username" name="username" placeholder="Ваше имя" required>
    
    <label for="email">Email:</label>
    <input type="email" id="email" name="email" placeholder="example@mail.com" required>
    
    <label for="gender">Пол:</label>
    <input type="radio" name="gender" value="male" id="male">
    <label for="male">Мужской</label>
    <input type="radio" name="gender" value="female" id="female">
    <label for="female">Женский</label>
    
    <label for="hobbies">Хобби:</label>
    <input type="checkbox" name="hobbies" value="music"> Музыка
    <input type="checkbox" name="hobbies" value="sports"> Спорт
    <input type="checkbox" name="hobbies" value="reading"> Чтение
    
    <label for="country">Страна:</label>
    <select id="country" name="country">
      <option value="ru">Россия</option>
      <option value="us">США</option>
      <option value="uk">Великобритания</option>
    </select>
    
    <button type="submit">Зарегистрироваться</button>
  </fieldset>
</form>
Объяснение
<fieldset> и <legend> группируют поля формы и дают им заголовок.

<input type="radio"> позволяет выбрать только один вариант, <input type="checkbox"> — несколько.

<select> с <option> создаёт выпадающий список.

Атрибут required проверяет обязательность поля до отправки формы.

<form> с action и method отправляет данные на сервер.

Формы — это основной способ взаимодействия пользователя с веб-сайтом, без них невозможно создать регистрацию, авторизацию или опрос.
