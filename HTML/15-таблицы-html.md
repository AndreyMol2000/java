# Таблицы в HTML

## Краткая информация
Таблицы используются для **структурированного отображения данных** в виде строк и колонок.  
Основные элементы:  
- `<table>` — контейнер таблицы  
- `<tr>` — строка таблицы  
- `<td>` — ячейка данных  
- `<th>` — заголовочная ячейка  

---

## Пример простой таблицы

```html
<table border="1">
  <tr>
    <th>Имя</th>
    <th>Возраст</th>
    <th>Город</th>
  </tr>
  <tr>
    <td>Андрей</td>
    <td>25</td>
    <td>Москва</td>
  </tr>
  <tr>
    <td>Мария</td>
    <td>30</td>
    <td>Санкт-Петербург</td>
  </tr>
</table>
<th> выделяется жирным текстом и по умолчанию центрируется.

<td> содержит обычные данные.

Атрибут border задаёт границы (для быстрых примеров, в реальных проектах лучше использовать CSS).

Подробная информация
Атрибуты таблиц
border — граница таблицы (устаревший, лучше CSS).

cellpadding — отступ внутри ячеек (устаревший, лучше CSS padding).

cellspacing — расстояние между ячейками (устаревший, лучше CSS border-spacing).

width, height — размеры таблицы.

align — выравнивание таблицы (устаревший, лучше CSS).

Объединение ячеек
colspan — объединение нескольких колонок.

rowspan — объединение нескольких строк.

Объяснение
Таблицы идеально подходят для структурированных данных, но не для общего оформления страницы.

Используем семантические теги <th> для заголовков колонок.

Объединение ячеек с colspan и rowspan помогает создавать более сложные таблицы.

Для современного HTML стилизация через CSS предпочтительнее устаревших атрибутов (border, cellpadding, align).

Таблицы могут использоваться и для отображения финансовых данных, расписаний, прайсов и аналитики.
