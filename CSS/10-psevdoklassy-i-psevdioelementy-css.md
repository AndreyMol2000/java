# Псевдоклассы и псевдоэлементы в CSS

## Краткая информация
- **Псевдоклассы** — позволяют применять стили к элементам в зависимости от состояния или позиции  
- **Псевдоэлементы** — позволяют создавать виртуальные элементы внутри блока для стилизации части контента  

---

## Примеры: Псевдоклассы

```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Псевдоклассы пример</title>
    <style>
        a {
            color: blue;
            text-decoration: none;
        }

        a:hover {
            color: red; /* при наведении */
        }

        li:first-child {
            font-weight: bold; /* первый элемент списка */
        }

        li:last-child {
            color: green; /* последний элемент списка */
        }

        li:nth-child(2) {
            font-style: italic; /* второй элемент списка */
        }
    </style>
</head>
<body>
    <ul>
        <li>Элемент 1</li>
        <li>Элемент 2</li>
        <li>Элемент 3</li>
    </ul>
    <a href="#">Ссылка</a>
</body>
</html>
Примеры: Псевдоэлементы
html
Копировать код
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Псевдоэлементы пример</title>
    <style>
        p::before {
            content: "Начало: "; /* вставка перед содержимым */
            font-weight: bold;
        }

        p::after {
            content: " :Конец"; /* вставка после содержимого */
            color: red;
        }

        span::first-letter {
            font-size: 2em; /* первый символ увеличен */
            color: blue;
        }
    </style>
</head>
<body>
    <p>Пример текста</p>
    <p><span>Текст с увеличенной первой буквой</span></p>
</body>
</html>
Подробная информация
Псевдоклассы
:hover — при наведении мыши

:active — при нажатии

:focus — при фокусе на элементе (input, textarea)

:first-child / :last-child / :nth-child(n) — выборка по позиции

:not(selector) — исключение элементов

Псевдоэлементы
::before — вставка контента перед элементом

::after — вставка контента после элемента

::first-letter — стилизация первой буквы

::first-line — стилизация первой строки

Объяснение
Псевдоклассы и псевдоэлементы позволяют сильно сократить HTML, добавляя стили без лишних тегов

Используются для эффектов наведения, вставки декоративных элементов и улучшения визуального оформления

В комбинации с анимациями и переходами повышают интерактивность интерфейса
